# Pi Comms 领域语言

Pi Comms 让多个用户及其 Pi Agent 在群组中共同交流和协作。本文件定义产品界面与设计文档统一使用的核心语言。

## Language

**群组**：
用户和 Agent 进行公开交流的会话集合，是用户创建、发现、加入和管理的主要对象。
_Avoid_: 协作空间、聊天室

**附近群组**：
当前网络中的其他设备允许加入的群组。群组名称和在线人数可以在加入前显示。
_Avoid_: 附近协作空间、附近 Broker

**本机群组**：
数据保存在当前设备上的群组，可以仅供本机使用，也可以允许附近设备加入。
_Avoid_: 本机空间

**群主**：
创建群组并管理该群组的本机 Pi Session 身份。群主身份不转让，也不等同于远程服务器管理员。
_Avoid_: Broker 管理员、房主

**长期成员**：
首次加入群组后获得稳定成员身份的用户。短暂离线、Session 重开或网络变化不会取消该身份。
_Avoid_: 临时连接、访客

**附近加入**：
群组允许当前普通网络中的其他设备发现并加入的可见范围。
_Avoid_: 开放 Broker、创建协作空间

**Broker**：
承载群组、消息和成员连接的内部通信服务，不是用户需要选择或理解的产品对象。
_Avoid_: 在用户界面中使用 Broker、把 Broker 称为协作空间

**群组邀请码**：
群主创建附近群组时可以启用的短准入凭证。默认不启用；启用后，首次成为该群长期成员时需要输入。每个群组独立生成，不代表其他群组的访问权。
_Avoid_: 设备邀请码、共享邀请码、密码、密钥

**群主凭证**：
让创建群组的本机 Pi Session 恢复群主管理身份的凭证。恢复管理权会使旧凭证失效。
_Avoid_: 账号、管理员密码
