# 阶段 8：权限系统

目标：为每个 Agent 提供自动接收、需要批准和禁止接收三种模式。

## 任务

- [x] 定义并保存 Agent 接收权限状态。
- [x] 实现“自动接收”，并将其设为默认模式。
- [x] 实现“需要批准”。
- [x] 实现“禁止接收”。
- [x] 在 TUI 中提供权限切换入口。
- [x] 为“需要批准”实现待批准列表。
- [x] 在成员列表中显示 Agent 待批准状态。
- [x] 实现批准操作，批准后再注入 Agent。
- [x] 实现拒绝操作。
- [x] 对禁止、拒绝和无效请求返回失败状态。
- [x] 将失败状态公开显示在群聊中。
- [x] 路由前同时检查目标在线状态和权限。

## 完成条件

- [x] 三种模式均可切换并按定义生效。
- [x] 未批准的请求不会注入 Agent。
- [x] 被禁止或拒绝的请求有明确的公开状态。
